# Minecraft Top-Down Biome Images (19 Classes)

A curated collection of top-down Minecraft biome images arranged into 19 class folders. Built for image classification benchmarks, prototyping, and educational ML projects.

## Highlights
- Format: PNG top-down screenshots
- Size: ~5 GB total
- Classes: 19 biomes (folder names are labels)
- Layout: `<biome_name>/*.png`

## Generation / Source
Collected via two bots (one fully automatic, one semi-automatic) that captured top-down screenshots. Version 1.21, no texture pack (vanilla), no shader pack. Camera height varies; mix of HD and 2K images; includes both day and night shots.
